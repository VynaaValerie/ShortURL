URL Shortener API Documentation

Integrate our URL shortening service into your applications

API Endpoints

POST /api/shorten
Create a new short URL. You can optionally specify a custom name for the URL.
Request
Content-Type: application/json
Parameter Type Required Description
url string Yes The long URL to be shortened
customName string No Custom name for the short URL (must be unique) 
{
  "url": "https://example.com/very-long-url",
  "customName": "my-link"
}
Response (Success)
{
  "success": true,
  "data": {
    "id": "V1StGXR8_Z5jdHi6",
    "originalUrl": "https://example.com/very-long-url",
    "shortUrl": "yourdomain.com/my-link",
    "shortCode": "my-link",
    "createdAt": "2023-07-20T12:34:56.789Z"
  }
}
Response (Error)
{
  "success": false,
  "error": "Custom name is already taken"
}
GET /api/info/:code
Get information about a short URL, including the original URL and click statistics.
Request
URL Parameter: code (the short code)
Response (Success)
{
  "success": true,
  "data": {
    "id": "V1StGXR8_Z5jdHi6",
    "originalUrl": "https://example.com/very-long-url",
    "shortUrl": "yourdomain.com/my-link",
    "shortCode": "my-link",
    "createdAt": "2023-07-20T12:34:56.789Z",
    "clicks": 42
  }
}
Response (Error)
{
  "success": false,
  "error": "Short URL not found"
}
GET /:code
Redirect to the original URL associated with the short code.
Request
URL Parameter: code (the short code)
Response
HTTP 301 redirect to the original URL
