API Documentation

Integrate ShortURL with your applications

Shorten a URL

Create a short URL from a long URL

POST
/api/shorten

Request

{
  "originalUrl": "https://example.com/very/long/url/to/be/shortened"
}

Response

{
  "originalUrl": "https://example.com/very/long/url/to/be/shortened",
  "shortUrl": "https://yourdomain.com/abc123"
}

Response Codes

200 OK
URL shortened successfully
400 Bad Request
Missing or invalid URL
500 Server Error
Internal server error

Redirect Short URL

Access a short URL to be redirected to the original URL

GET
/{shortCode}

Example

GET https://yourdomain.com/abc123

Response Codes

301 Moved
Redirect to original URL
404 Not Found
Short URL not found
